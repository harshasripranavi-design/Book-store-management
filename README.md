# Book Store Management System

## Project Overview
The Book Store Management System is a web-based application that allows users to browse, search, and purchase books online.

## Features
- User Registration and Login
- Browse Books
- Search Books
- Add to Cart
- Place Orders
- Admin Dashboard

## Technologies Used
- HTML
- CSS
- JavaScript
- Python
- MySQL

## User Flow
Login → Browse Books → Add to Cart → Checkout → Order Confirmation

## Future Scope
- Online Payments
- Mobile App Support
- AI Book Recommendations
